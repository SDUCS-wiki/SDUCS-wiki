    Anaconda的安装与使用¶

    为了不给使用带来困扰，本 Wiki 不推荐电脑用户名中带有任何中文

    1. 安装¶

    1.1 下载¶

    • 前往 Anaconda 官网下载 Anaconda Individual Edition (Anaconda个人版)

    1.2 安装¶

    • 找到下载好的 Anaconda3-2021.05-Windows-x86_64.exe，启动该文件

    • 按照以下步骤进行安装

    • 关键安装步骤已经完成，剩下步骤可一路Next/Finish

    1.3 系统环境变量配置¶

    • 单击Windows徽标键，在键盘上输入path

    • 点击 环境变量，在系统变量中找到Path，双击Path，点击浏览

    • 依次找到下述文件夹，并添加，然后确认保存并重启
    1

    1

    2. 使用Conda¶

    2.1 创建环境¶

    创建一名称为name且Python版本为3. 7的Conda环境

    1
    conda create -n name python=3.7

    2.2 销毁环境¶

    删除一名称为name的Conda环境

    1
    conda remove -n name --all

    2.3 进入环境¶

    进入一名称为name的Conda环境

    1
    conda activate name

    2.4 退出环境¶

    1
    conda deactivate

    2.5 安装包¶

    1
    conda install xxx

    2.6 查看当前环境Python版本¶

    1
    python --version　　# 查看python解释器的版本
